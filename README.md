# Primer Parcial - Integración Continua con GitHub Actions y Surge.sh

Descripción del Proyecto

Este proyecto fue desarrollado como parte del Primer Parcial, con el objetivo de implementar un flujo básico de Integración Continua (CI/CD) utilizando GitHub Actions y Surge.sh.

La aplicación consiste en una página web estática desarrollada en HTML, cuyo despliegue se realiza de manera automática cada vez que se envían cambios al repositorio principal. Gracias a la automatización proporcionada por GitHub Actions, el sitio se publica instantáneamente en Surge.sh sin necesidad de intervención manual.

Objetivos Alcanzados
Creación y gestión de un repositorio Git local.
Publicación del proyecto en GitHub.
Implementación de GitHub Actions para automatizar el despliegue.
Uso de Secrets de GitHub para proteger credenciales sensibles.
Despliegue continuo hacia Surge.sh.
Desarrollo de una página web personalizada.
